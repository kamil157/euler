# Project Euler
Solutions to tasks from https://projecteuler.net/

### To install dependencies:
`pip3 install -r requirements.txt`

### To run tests:

from project root run:

```
export PYTHONPATH=\`pwd\`
cd tests
python3 -m unittest
```

### To run a single task:

from project root run:

```
export PYTHONPATH=\`pwd\`
cd tasks
python3 task1.py
```
